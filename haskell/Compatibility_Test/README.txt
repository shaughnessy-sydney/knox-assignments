READ ME

This compatibility tester calculates two individuals romantic compatibility based on their names' as a string ASCII int value. It provides a percentage and star rating.
The concept of the code is adapted from the Tomodachi Life compatibility tester. 
(While I initially intended to create the dinosaur jumping game, I could not get the frames to update, and I got too frustrated.)

INSTALLATION INSTRUCTIONS
A Haskell environment is required to run this program
Download GHC via https://www.haskell.org/ghcup/
Install GHC through Command Prompt: ghcup install ghc
Install Cabal after: ghcup install cabal
Clone the repo: git clone https://github.com/Knox-College-Computer-Science/haskell-project-shaughnessy-sydney
cd haskell-project-shaughnessy-sydney
Compile the code: cabal build
Run the code: cabal run


LIMITATIONS
1) Can't test more than two names at a time.
2) The results are arbitrary and use no real factors.
3) Lacks GUI
